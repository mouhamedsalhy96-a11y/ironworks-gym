# Ironworks Gym - Modern Fitness E-Commerce Platform

A high-performance, dark-themed gym website built to demonstrate modern frontend development, UI/UX design principles, and secure payment integrations. Designed for a premium fitness brand offering memberships and personal training services.

## 🚀 Live Demo

*https://ironworks-gym-gamma.vercel.app*

## 🛠 Tech Stack

*   **Framework:** Next.js (App Router)
*   **Styling:** Tailwind CSS (v4)
*   **Icons:** Lucide React & Custom SVGs
*   **Payments:** Stripe Checkout API
*   **Mapping:** Google Maps Embed API

## ✨ Key Features

*   **Responsive Modern UI:** A fully responsive, high-contrast dark theme designed to evoke a premium, "hardcore" gym aesthetic.
*   **Stripe E-Commerce Integration:** Functional checkout routing for Day Passes, Monthly Subscriptions, and Annual Memberships securely handled via Stripe's API.
*   **Interactive Components:** Smooth scrolling, hover states, and dynamic pricing cards built with Tailwind utility classes.
*   **Lead Generation:** Fully styled Personal Training inquiry form with success-state routing.
*   **Modular Architecture:** Clean, component-based file structure using Next.js App Router best practices.

## 🚦 Getting Started

To run this project locally, you will need Node.js installed on your machine.

**1. Clone the repository**
\`\`\`bash
git clone https://github.com/your-username/ironworks-gym.git
cd ironworks-gym
\`\`\`

**2. Install dependencies**
\`\`\`bash
npm install
\`\`\`

**3. Set up your environment variables**
Create a \`.env.local\` file in the root directory and add your Stripe API keys:
\`\`\`env
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_YOUR_KEY_HERE
STRIPE_SECRET_KEY=sk_test_YOUR_KEY_HERE
\`\`\`

**4. Run the development server**
\`\`\`bash
npm run dev
\`\`\`
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
